# Si aparece “servidor no disponible”

## Lo más probable

Estás abriendo la dirección local antigua:

http://127.0.0.1:4173

Esa dirección solo funciona en el PC donde Igor arrancó el servidor local.

En iPhone/iPad u otro ordenador NO sirve.

## Dirección correcta ahora

La PWA publicada en GitHub Pages es esta:

https://lyux69.github.io/powertest-luis-pwa/

Esa es la que debes abrir desde móvil, iPad o cualquier PC.

## Comprobación hecha por Igor

La URL pública responde correctamente:

- GitHub Pages: activo.
- HTTP 200.
- La página carga `PowerTest Luis`.
- Supabase responde desde la página pública.

## Qué hacer

1. Cierra la pestaña antigua.
2. Abre Safari/Chrome.
3. Escribe exactamente:

https://lyux69.github.io/powertest-luis-pwa/

4. Si sale algo raro, recarga.
5. En iPhone/iPad, luego usa:

Compartir → Añadir a pantalla de inicio

http://127.0.0.1:4173
